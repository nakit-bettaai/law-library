# พระราชบัญญัติการรักษาความมั่นคงปลอดภัยไซเบอร์ พ.ศ. 2562
# Cybersecurity Act B.E. 2562

## วัตถุประสงค์ / Purpose
กำหนดมาตรการและกลไกในการรับมือกับภัยคุกคามทางไซเบอร์ โดยเฉพาะในโครงสร้างพื้นฐานที่สำคัญ / Establish measures and mechanisms to deal with cyber threats, especially in critical infrastructure.

---

## โครงสร้างพื้นฐานสำคัญทางสารสนเทศ / Critical Information Infrastructure (CII)
หน่วยงานและองค์กรใน 8 ด้านต้องปฏิบัติตาม / Organizations in 8 sectors must comply:
1. ความมั่นคงของรัฐ / National security
2. บริการภาครัฐที่สำคัญ / Essential public services
3. การเงินการธนาคาร / Finance and banking
4. เทคโนโลยีสารสนเทศและโทรคมนาคม / IT and telecommunications
5. การขนส่งและโลจิสติกส์ / Transportation and logistics
6. พลังงานและสาธารณูปโภค / Energy and utilities
7. สาธารณสุข / Public health
8. อื่นๆ ตามที่คณะกรรมการกำหนด / Others as determined by the Committee

---

## หน้าที่ของหน่วยงาน CII / CII Organization Obligations
- จัดทำแผนรับมือเหตุการณ์ไซเบอร์ (Incident Response Plan) / Prepare a Cyber Incident Response Plan
- ปฏิบัติตามมาตรฐานความมั่นคงปลอดภัยที่กำหนด / Comply with prescribed security standards
- แจ้งเหตุการณ์ไซเบอร์ภายใน 72 ชั่วโมง / Report cyber incidents within 72 hours
- ให้ความร่วมมือกับ สกมช. ในการตรวจสอบ / Cooperate with NCSA investigations

---

## ระดับภัยคุกคาม / Threat Levels
| ระดับ / Level | คำอธิบาย / Description | มาตรการ / Measures |
|---|---|---|
| ปกติ / Normal | ไม่มีภัยคุกคาม / No threat | มาตรฐานทั่วไป / Standard measures |
| เฝ้าระวัง / Alert | มีความเสี่ยง / Risk detected | เพิ่มการตรวจสอบ / Enhanced monitoring |
| วิกฤต / Critical | ภัยคุกคามร้ายแรง / Serious threat | ระดมกำลังเต็มที่ / Full mobilization |

---

## อำนาจพิเศษในภาวะวิกฤต / Special Powers in Crisis
เมื่อมีภัยคุกคามระดับวิกฤต สกมช. มีอำนาจ / When critical threat level, NCSA has power to:
- เข้าถึงระบบคอมพิวเตอร์ที่เกี่ยวข้อง / Access related computer systems
- สั่งให้หยุดการทำงานของระบบที่เป็นภัย / Order suspension of threatening systems
- ประสานกับต่างประเทศในการรับมือ / Coordinate with foreign entities

---

## บทลงโทษ / Penalties
- ไม่แจ้งเหตุภายในเวลาที่กำหนด: ปรับไม่เกิน 200,000 บาท / Failure to report timely: up to 200,000 THB fine
- ไม่ให้ความร่วมมือกับเจ้าหน้าที่: ปรับไม่เกิน 300,000 บาท / Non-cooperation: up to 300,000 THB fine
- ขัดขวางการปฏิบัติหน้าที่: จำคุกไม่เกิน 1 ปี หรือปรับไม่เกิน 100,000 บาท / Obstruction: up to 1 year imprisonment or 100,000 THB fine

---

## ความสัมพันธ์กับ PDPA / Relationship with PDPA
กฎหมายทั้งสองฉบับเสริมกัน / Both laws complement each other:
- PDPA: คุ้มครองข้อมูลส่วนบุคคล / Protects personal data
- Cybersecurity Act: คุ้มครองโครงสร้างพื้นฐาน / Protects infrastructure
- การละเมิดความปลอดภัยไซเบอร์อาจนำไปสู่ความรับผิดตาม PDPA ด้วย / Cyber breach may also trigger PDPA liability
